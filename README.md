# Fadma's Portfolio

Personal portfolio built with Next.js, TypeScript, and React.

## Live Link

[https://portfolio1-delta-gules-52.vercel.app](https://portfolio1-delta-gules-52.vercel.app)

## Stack

- Next.js
- React
- TypeScript
- Vitest
- Testing Library

## Local Development

```bash
npm install
npm run dev
```

Add your Formspree endpoint in `.env`:

```env
NEXT_PUBLIC_FORMSPREE_ENDPOINT=https://formspree.io/f/your-form-id
```
